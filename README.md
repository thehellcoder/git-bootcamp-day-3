# taskmanager

Учебный CLI-менеджер задач на Python. Позволяет добавлять задачи, просматривать список и помечать их как выполненные.

Минимальный API в `tasks.py` **расширен**: `add_task(title)`, `list_tasks()`, `complete_task(index)`, `delete_task(index)`.

## Установка

На данном этапе код можно запускать без виртуальной среды, т.к. используется только стандартная библиотека Python.
Склонируйте репозиторий и погнали!

```bash
git clone https://github.com/thehellcoder/git-bootcamp-day-3.git
cd git-bootcamp-day-3
```

## Запуск

```bash
python3 -c "import tasks; tasks.add_task('купить хлеб'); tasks.list_tasks()"
```
